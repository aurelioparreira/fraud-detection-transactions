# Detecção de Anomalias em Transações

Projeto em Python para detectar fraudes em transações financeiras usando **scikit-learn** e técnicas de balanceamento de classes.  
O trabalho foi desenvolvido em Google Colab e organizado para execução local ou em nuvem.

---

## 📊 Dataset
O dataset utilizado está disponível publicamente:  
[Credit Card Fraud Detection Dataset](https://storage.googleapis.com/download.tensorflow.org/data/creditcard.csv)

Ele contém transações de cartão de crédito europeias em setembro de 2013, com 284.807 registros, dos quais apenas 492 são fraudes (classe altamente desbalanceada).
